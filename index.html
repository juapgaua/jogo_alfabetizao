BRINCANDO COM O ALFABETO

Neste jogo de alfabetização, você deve escolher uma letra — clicando nela ou usando o botão “Sortear letra”. Quando letra é selecionada, escreva uma palavra que comece com aquela letra. Continue até utilizar todas as letras disponíveis. Se quiser recomeçar, basta clicar no botão “Reiniciar” para limpar tudo e exibir as letras novamente.
