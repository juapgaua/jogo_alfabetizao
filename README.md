# Jogo Interativo de Alfabetização (A–Z)

Este projeto é uma aplicação web desenvolvida em **HTML, CSS e JavaScript** para auxiliar pessoas em processo de alfabetização.  
A proposta é trabalhar o reconhecimento das letras do alfabeto e estimular a escrita de palavras por meio de interação direta com a interface.

## Como funciona

- Todas as letras do alfabeto (A–Z) aparecem como botões.  
- Ao clicar em uma letra **ou sortear uma aleatoriamente**, o campo correspondente recebe foco.  
- A letra escolhida desaparece da lista para evitar repetição.  
- O usuário escreve uma palavra que comece com a letra selecionada.  
- Um botão **“Reiniciar”** limpa todos os campos e restaura as letras.

## Recursos

- 26 campos de entrada (A a Z)  
- Botões interativos para cada letra  
- Botão "Sortear letra"  
- Botão "Reiniciar"  
- Foco automático no campo correspondente  
- Código organizado em HTML, CSS e JS separados  
- Ideal para alfabetização inicial
